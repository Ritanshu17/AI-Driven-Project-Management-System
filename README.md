# AI-Driven-Project-Management-System
# 🚀 Engineering Workspace

> An AI-powered engineering collaboration platform designed to help software teams plan, build, and ship projects faster.

Engineering Workspace combines project management, team collaboration, and AI-powered productivity into a single modern web application. Inspired by tools like **Linear**, **Jira**, **ClickUp**, **Slack**, **Discord**, **Loom**, and **Otter.ai**, the goal is to provide developers and engineering teams with one unified workspace.

> ⚠️ This project is currently under active development.

---

# ✨ Features (Current Phase)

- User Authentication UI
- Dashboard
- Project Management
- Kanban Board
- Drag & Drop Tasks
- Light / Dark Theme

- Analytics Dashboard
- Local JSON Data (Frontend Only)

# 🚀 Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/Ritanshu17/AI-Driven-Project-Management-System.git

cd engineering-workspace/frontend
```

---

## 2. Install Dependencies

```bash
npm install lucide-react zustand recharts @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities next-themes
```

---

## 3. Run Development Server

```bash
npm run dev
```

Open your browser:

```
http://localhost:3000
```
